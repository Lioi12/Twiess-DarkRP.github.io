Правила Gmod-Сервера
FuriousRP

Общие правила:

1.1 RDM - убийство без причины, на сервере не существует понятие как (FreeKill).➟(Ban 50 min)
1.2 PropPush - использование пропа/объекта с целью сдвинуть кого-либо.➟(Ban 50 min)
1.3 PropSurf - использование пропа/объекта в виде летательного средства. ➟ (Ban 50 min)
1.4 PropClimb - использование пропа/объекта для преодоление физического препятствия.➟(Ban 50 min)
1.5 FearRp -(Правило Страха) отсутствие страха у игрока,стоящего перед прямой угрозой жизни.➟(Ban 50 min)
1.6 NonRPGun - нарушение установленного [Разрешенного Оружия].➟(Ban 50 min)
1.7 DoorAbuse - злоупотребление функцией двери [Открытие/Закрытие].➟(Ban 50 min)
1.8 JobAbuse - взятие профессии для своей выгоды, избегая рп процесса.➟(Ban 50 min)
1.9 VanishProps - невидимые пропы мешающие рп действию.➟(Ban 50 min)
1.10 BagAbuse - использования бага для выгоды игрока или помехе рп процессу, (случайное применение не считается).➟(Ban 1 day - 1 week)
1.11 NonRp - действия не связанные с РП отыгровкой.➟(Ban 50 min)
1.12 NewLifeRule (NLR) - использование информации из прошлой жизни.➟(Ban 50 min)
1.13 FreeWarrent - розыск без [адекватной] причины. ➟(Ban 50 min)
1.14 BodyBlock - умышленное закрытие прохода своим телом с целью ограничения передвижения.➟(Ban 50 min)
1.15 PropBlock - закрытие пропами проходов,препятствующих передвижению.➟(Ban 50 min)
1.16 TeamKill - убийство игрока из своей фракции / альянса / организации.➟(Ban 50 min)
1.17 FreeDamage - нанесение урона по игроку без причины.➟(Ban 50 min)
1.18 FadingDoorAbuse - закрытие FD перед человеком не давая тому пройти/выйти ради своей выгоды.➟(Ban 50 min)
1.19 FreeDemote - увольнение без [адекватной] причины.➟(Ban 50 min)
1.20 MetaGaming - использование NonRp каналов связи (ООС) для передачи важной РП информации.➟(Ban 50 min)
1.21 FreeArrest - арест без [адекватной] причины.➟(Ban 50 min)
1.22 CustomJob - изменение названия профессии,не связанной с данной работой.➟(Ban 30 min)
1.23 AdvertAbuse - чрезмерное отправление одинаковых сообщений в (Advert).➟(Mute 5 min){От 2 сообщений}
1.24 MassRDM - многократное убийство без причины.➟(Ban 3 hour){От 3 игроков}
1.25 MassTeamKill - многократные убийства игроков из своей фракции / альянса / организации.➟(Ban 2 hour){От 3 игроков}
1.26 PowerGaming - преувеличение своих возможностей в военном деле.➟(Ban 50 min)
